# ApiPilot

ApiPilot is a powerful API Client for VS Code.

## Features

- **Request Editor**: Create and manage HTTP requests.
- **Environment Management**: Manage environments and variables.
- **History**: View request history.
- **Collections**: Organize requests into collections.
- **Code Generation**: Generate code snippets for various languages.
- **Import/Export**: Import from Postman, OpenAPI, and cURL.

## Usage

1. Open the ApiPilot sidebar.
2. Create a new request.
3. Send the request and view the response.

## Development

1. Run `npm install` to install dependencies.
2. Run `npm run dev:webview` to start the webview development server.
3. Press `F5` to start debugging the extension.
